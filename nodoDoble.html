nodoDoble()

Clase nodoDoble: probee de un modelo para la configuración de los nodos de la clase lista, contiene un dato numerico para le indicador y tres ligas distintas

Constructor

new nodoDoble()

construye el nodo, donde indicador es un valor numérico que indica si el nodo es serpiente, escalera o genérico, recibe 0 para serpiente, 1 para escalera y null para el default. LInd es una liga que enlaza la casilla con el nodo donde finaliza la escaler ao la serpiente, null para el default. LCons es una liga que enlaza el nodo con el directamente siguiente.
